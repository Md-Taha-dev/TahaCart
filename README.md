# TahaCart E-Commerce Web Application

A robust full-stack e-commerce platform built using Python, Flask, and a relational MySQL backend database. 

## 🚀 Features
- **Secure Authentication**: User signup, login, and sessions handled securely via Flask-Login and Bcrypt password hashing.
- **Dynamic Cart Pipelines**: Seamlessly add, remove, increase, and decrease item quantities with automated database inventory synchronization.
- **Admin Command Center**: Role-based access control panel to view system users, track orders inventory, add new products, and update delivery statuses via dynamic states.

## 🛠️ Tech Stack
- **Backend Architecture**: Python, Flask, SQLAlchemy ORM
- **Database Storage**: MySQL Relational Database
- **Frontend Presentation**: HTML5, CSS3 (CSS Grid/Flexbox Layouts)
